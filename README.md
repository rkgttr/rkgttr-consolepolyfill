### How to Install

```sh
$ npm install rkgttr-consolepolyfill --save-dev
```
or

```sh
$ yarn add rkgttr-consolepolyfill --dev
```

### Getting Started

```js
import consolePolyfill from 'rkgttr-consolepolyfill';
```

or

```js
require('rkgttr-arrayfrompolyfill');
```
